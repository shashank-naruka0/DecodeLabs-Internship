# DecodeLabs-Internship
This project focuses on analyzing end-to-end supply chain performance by transforming raw operational data into meaningful insights using Power Query, Data Modeling, and Advanced DAX. It helped me understand how analytics can support better decision-making in areas such as inventory management, logistics efficiency, and vendor performance.<br>
<br>
🚀 Project Objective<br>
To analyze sales data and convert raw information into actionable business insights using Excel.<br>
<br>
🎯 Key Insights<br>
- 📉 Revenue shows a declining trend over time.<br>
- 📈 Instagram is the top-performing referral source.<br>
- 🛒 High-performing products: Printer, Chair, Laptop.<br>
- ⚠️ High cancellation and return rates observed.<br>
- 💳 Payment methods are evenly distributed.<br>
<br>
